# Grid Map

miniMUGA genetic and physical map for mm39 genome assembly.

## Usage

``` r
data(gridfile_mini)
```

## Format

A 11125 x 3 data frame with marker names included as row names:

- chr:

  Chromosome name

- pos:

  Marker position in basepairs

- cM:

  Marker position in centiMorgan
