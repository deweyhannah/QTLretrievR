# Grid Map

Genetic and Physical Maps to be used with genotypes from Genotyping by
lcWGS.

## Usage

``` r
data(gridfile_quilt)
```

## Format

A 999999 x 3 data frame with marker names included as row names:

- chr:

  Chromosome name

- pos:

  Marker position in basepairs

- cM:

  Marker position in centiMorgan

## Source

See https://www.biorxiv.org/content/10.1101/2025.05.19.654056v1 for more
information on Genotyping by lcWGS.
