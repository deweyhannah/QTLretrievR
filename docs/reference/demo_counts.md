# Demo Counts

Downsampled mESC gene expression counts to be used for vignette.

## Usage

``` r
data(demo_counts)
```

## Format

A 1000 x 10 matrix of normalized gene expression with each column as a
different sample and genes in the rows.

- rows:

  Gene names as Ensembl IDs

- cols:

  Individual sample IDs

## Source

Skelly et al., Mapping the Effects of Genetic Variation on Chromatin
State and Gene Expression Reveals Loci That Control Ground State
Pluripotency, Cell Stem Cell (2020),
https://doi.org/10.1016/j.stem.2020.07.005
