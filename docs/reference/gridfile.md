# Grid Map

75k genetic and physical map for mm39 genome assembly.

## Usage

``` r
data(gridfile)
```

## Format

A 74165 x 4 data frame with marker names included as row names:

- chr:

  Chromosome name

- pos:

  Marker position in basepairs

- cM:

  Marker position in centiMorgan
