# Grid Map

Genetic and Physical Maps to be used with genotypes from the JAX
lcwgs-hr nextflow pipeline.

## Usage

``` r
data(gridfile250)
```

## Format

A 250000 x 3 data frame with marker names included as row names:

- chr:

  Chromosome name

- pos:

  Marker position in basepairs

- cM:

  Marker position in centiMorgan
