# Grid Map

GigaMUGA genetic and physical maps for mm39 genome assembly.

## Usage

``` r
data(gridfile_GM)
```

## Format

A 143259 x 3 data frame with marker names included as row names:

- chr:

  Chromosome name

- pos:

  Marker position in basepairs

- cM:

  Marker position in centiMorgan
