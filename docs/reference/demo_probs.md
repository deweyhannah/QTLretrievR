# Demo Probabilities

Sample genotype probabilities to be used for vignette.

## Usage

``` r
data(demo_probs)
```

## Format

A list of genoprobs for each sample/marker separated by chromosome

- Outer List:

  Chromosome

- Inner List:

  3D dataset with samples x haplotype x marker

## Source

Skelly et al., Mapping the Effects of Genetic Variation on Chromatin
State and Gene Expression Reveals Loci That Control Ground State
Pluripotency, Cell Stem Cell (2020),
https://doi.org/10.1016/j.stem.2020.07.005
