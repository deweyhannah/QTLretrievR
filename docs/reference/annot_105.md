# Annotations file

Ensembl v105 annotations.

## Usage

``` r
data(annot_105)
```

## Format

A 55416 x 5 data frame with gene annotations from Ensembl v105.

- id:

  Ensembl gene IDs

- symbol:

  Associated MGI symbol

- chr:

  Chromosome of the gene

- start:

  Start position (bp) on chromosome

- end:

  End position (bp) on chromosome

ensembl.org
